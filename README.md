# ShadowMail

**ShadowMail** is a secure, anonymous, and reactive temporary email service designed to protect your privacy. Whether you need to sign up for a service, test a platform, or avoid spam, ShadowMail provides a reliable solution without requiring any personal data.

---

## 🌟 Key Features

*   **Secure & Anonymous:** No personal identification required. Protect yourself from spammers and potential data breaches.
*   **Persistent Accounts:** Unlike many services, your ShadowMail account and generated addresses **do not self-delete**. 
*   **Multi-Account Support:** Manage up to **12 unique email addresses** simultaneously under one account.
*   **Auto-Cleanup:** Emails received in your inbox are automatically deleted after **10 days**, ensuring your storage remains clean without manual intervention.
*   **Reactive UI:** Built with Svelte for a lightning-fast, smooth user experience.

---

## 🛠 Tech Stack

*   **Frontend:** [Svelte](https://svelte.dev/) for a responsive and high-performance interface.
*   **Backend:** [Node.js](https://nodejs.org/) with [Express](https://expressjs.com/) for robust API handling.

## 🚀 Getting Started

### Prerequisites
Ensure you have the following installed on your machine:
*   [Node.js](https://nodejs.org/) (v22+)
*   npm

---

## 💡 How It Works

1.  **Register:** Create an account anonymously without providing personal info.
2.  **Generate:** Create up to 12 temporary email addresses.
3.  **Receive:** Use your addresses anywhere.
4.  **Manage:** Access your inbox at any time; emails are stored for 10 days before automatic deletion.

---

## 🛡 Security Policy
ShadowMail is committed to privacy. We do not track users or store identifiable information. Because we utilize temporary email addresses, we recommend using ShadowMail for low-stakes services and avoiding use for critical account recovery where permanent access is required.
